# INEFFICIENCY DEFINITION

An inefficiency exists when the same economic exposure
has different values in different places, times,
instruments or forms.

Categories

1. Geographic

Example:
India Steel vs Dubai Steel

2. Time

Example:
Spot vs Futures

3. Instrument

Example:
Physical Gold vs Gold Futures

4. Currency

Example:
USD Asset vs INR Asset

5. Freight

Example:
Cargo Route A vs Cargo Route B

6. Liquidity

Example:
Same asset, different market depth

7. Tax

Example:
Different tax treatment creates spread

8. Inventory

Example:
Owned inventory cheaper than market

Goal

Detect inefficiency.

Calculate executable profit.

Rank opportunities.

Execute or take delivery.
