# PROJECT NORTH STAR

Mission

Build a Universal Market Inefficiency Operating System.

Primary Objective

Find, rank and execute inefficiencies between markets.

Priority Order

1. Arbitrage
2. Relative Value
3. Delivery Fallback
4. Inventory Optimization

Not Objectives

- Predict market direction
- Build a retail trading platform
- Build charting software
- Build a brokerage
- Build a discretionary trading system

Success Metric

The system consistently discovers profitable opportunities after all costs, taxes, brokerage, slippage, financing and settlement considerations.
